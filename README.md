# sakeidol
個人でアイドルマスターのアイドルの名前を関する酒を飲んだ記録です。

## 記録する基準について
- 銘柄と名前が完全一致するものが記録されているとは限りません。**選出基準はすごく適当です**。
    - 都道府県もアイドルの出身地だとも限りません。一致してたらうれしいなーくらいの気持ちです。
